🚀 JobHunt.track — Job Application Tracker

A sleek, modern, and fully client-side job application tracker designed to help you stay organized during your job hunt. Track applications, monitor progress, manage resumes, and never miss a follow-up again.

✨ Features
📊 Smart Dashboard

Real-time stats (Total, Applied, Interview, Offers)

Weekly application tracker

Visual pipeline overview

Response rate indicator

📋 Application Management

Add, edit, and delete job applications

Track:

Company

Role

Status (Applied, Interview, Follow-up, Offer, Rejected)

Salary

Location

Job type (Full-time, Remote, etc.)

Priority system (⭐ 1–5)

⚡ Follow-up Intelligence

Automatically detects applications with no response after 7 days

Highlights them for quick action

🤖 AI Auto-Fill (Experimental)

Paste a job link

Auto-extract:

Company

Role

Location

Salary (if available)

Job description summary

📎 File Management

Upload and manage:

Resume (PDF/DOC/DOCX)

Cover letter

Built-in preview for PDFs

Download anytime

🔍 Search & Filters

Search across company, role, notes, and location

Filter by status

Sort by:

Date

Priority

Company

Days since applied

🧩 Views

📑 Table view (detailed)

🗂 Card view (compact & visual)

💾 Data Control

Local storage (no backend required)

Export all data as CSV

Backup reminders after 20+ applications

📝 Activity Timeline

Log updates for each application

Track your interview journey step-by-step

🛠️ Tech Stack

HTML5

CSS3 (Custom UI + Animations)

Vanilla JavaScript (No frameworks)

LocalStorage API (for persistence)

📦 Installation

No setup required.

Clone the repository:

git clone https://github.com/your-username/job-tracker.git

Open the file:

job-tracker.html

That’s it. Runs entirely in your browser.

🌐 Usage

Click “+ Add Application”

Paste a job link (optional AI autofill)

Fill details and save

Track progress from dashboard

⚠️ Notes

Data is stored in your browser using localStorage

Clearing browser data will erase your applications

AI auto-fill depends on external API (may require setup if extended)

🔒 Privacy

100% client-side

No data is sent to any server (except optional AI feature)

Your job data stays on your machine
